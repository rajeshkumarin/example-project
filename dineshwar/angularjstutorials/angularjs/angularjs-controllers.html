AngularJS Controllers

AngularJS applications are controlled by controllers.

AngularJS controllers are regular JavaScript Objects.

A controller is a JavaScript object containing attributes/properties and functions.

Each controller accepts $scope as a parameter which refers to the application/module that controller is to control.

In AngularJS, $scope is the application object (the owner of application variables and functions).

The controller creates two properties (variables) in the scope (firstName and lastName).

The ng-model directives bind the input fields to the controller properties (firstName and lastName).

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="myApp" ng-controller="myCtrl">

<b>Fst</b> Name: <input type="text" ng-model="firstName"><br><br>
<b>Lst</b></b> Name: <input type="text" ng-model="lastName">
<p><b>Its</b> Full Name: <b>{{firstName + " " + lastName}}</b></p>

</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
    $scope.firstName = "Sudhir";
    $scope.lastName = "Kushvaha";
});
</script>

</body>
</html>

Controller Methods

The example above demonstrated a controller object with two properties: lastName and firstName. A controller can also have methods (variables as functions):

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="myApp" ng-controller="personCtrl">

<b>Fst</b> Name:<input type="text" ng-model="firstName"><br><br>
<b>Mdl</b> Name: <input type="text" ng-model="middleName"><br><br>
<b>Lst</b> Name: <input type="text" ng-model="lastName"><br><br>

<p><b>It's</b> Full Name:<strong> {{fullName()}}<strong></p>

</div>

<script>
var app = angular.module('myApp', []);
app.controller('personCtrl', function($scope) {
    $scope.firstName = "Nishant";
    $scope.middleName = "Kumar";
     $scope.lastName = "Sharma";
    $scope.fullName = function() {
        return $scope.firstName + " " + $scope.middleName +" "+ $scope.lastName;
    };
});
</script>

</body>
</html>
