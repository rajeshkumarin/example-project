AngularJS Directives

AngularJS directives are used to extend HTML. These are special attributes starting with ng- prefix.

AngularJS has a set of built-in directives which offers functionality to your applications.

The ng-app This directive starts an AngularJS Application. an AngularJS application.

The ng-init This directive initializes application data.

The ng-model This directive binds the value of HTML controls (input, select, textarea) to application data.

The ng-repeat âˆ’ This directive repeats html elements for each item in a collection.

Example

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="" ng-init="firstName='Dineshwar'">

<p>Input something in the input box:</p>
<p>FirstName: <input type="text" ng-model="firstName"></p>
<p>You first name: {{ firstName }}</p>

<p ng-init="middleName='Kumar'">MiddleName: <input type="text" ng-model="middleName"></p>

<p>You "middle name: {{ middleName }}</p>

<p ng-init="lastName='Paswan'">LastName: <input type="text" ng-model="lastName"></p>

<p>You lastt name: {{ lastName }}</p>
</div>

</body>
</html>

Output:

Input something in the input box:

FirstName:

You first name: {{ firstName }}

MiddleName:

You "middle name: {{ middleName }}

LastName:

You lastt name: {{ lastName }}

Data Binding

The {{ firstName }} expression, in the example.Data binding in AngularJS binds AngularJS expressions with AngularJS data.

{{ firstName }} is bound with ng-model="firstName".

Example four text fields are bound together with four ng-model directives:

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div data-ng-app="" data-ng-init="quantity=110;price=5;invest=350;lose=130">

<h2>Cost Calculator</h2>

Quantity: <input class="label" type="number" ng-model="quantity">
Price: <input class="label" type="number" ng-model="price">
Invest:<input class="label" type="number" ng-model="invest">
Lose:<input class="label" type="number" ng-model="lose">
<p><b>Total in dollar:</b>{{quantity / price + invest- lose}}</p>

</div>

</body>
</html>

Repeating HTML Elements

The ng-repeat directive repeats an HTML element:

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="" ng-init="students=['Deepak is Best Boy','Harsh is Best Cricker','Tulshi is Best Actor','Rahim is Best Molana','Bipin is Best Officer','Rani is Best Dancer','Saweta is Best Teacher']">
  <p>Looping with ng-repeat:</p>
  <ul>
    <li ng-repeat="y in students">
      {{ y }}
    </li>
  </ul>
</div>

</body>
</html>

The ng-repeat directive actually clones HTML elements once for each item in a collection.

The ng-repeat directive used on an array of objects:

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="" ng-init="names=[{city:'Ranch',stat:'in Jharkhand'},
{city:'Patna',stat:'in Bihar'},
{city:'New Delhi',stat:'in Delhi'}]">

<p>Looping with objects:</p>
<ul>
  <li ng-repeat="x in names">
  {{ x.city + ', ' + x.stat}}</li>
</ul>

</div>

</body>
</html>
