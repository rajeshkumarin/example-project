AngularJS Forms

Forms in AngularJS provides data-binding and validation of input controls.

AngularJS enriches form filling and validation.

Input Controls

  • input elements
  • select elements
  • button elements
  • textarea elements

Data-Binding

Input controls provides data-binding by using the ng-model directive.

The application does now have a property named firstname. can be referred to in a controller: 

    <input type="text" ng-model="firstname">
	                
<!DOCTYPE html>
<html lang="en">
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="myApp" ng-controller="formCtrl">
  <form>
    First Name:<input type="text" ng-model="firstname">
    <p><b>It's First Name: </b><span ng-bind="firstname"></span></p>
  </form>
  <form>
	Middle Name:<input type="text" ng-model="middlename">
	<p><b>It's Middle Name: </b><span ng-bind="middlename"></span></p>
 </form>
  <form>
	Last Name:<input type="text" ng-model="lastname">
	<p><b>It's Last Name: </b><span ng-bind="lastname"></span></p>
 </form>
</div>

<script>
var app = angular.module('myApp', []);
app.controller('formCtrl', function($scope) {
    $scope.firstname = "Deepak";
     $scope.middlename = "Kumar";
	$scope.lastname = "Sinha";
});
</script>

</body>
</html>

Checkbox

A checkbox has the value true or false. Apply the ng-model directive to a checkbox,and use its value in your application. 

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="">
  <form>
    Check to show a pragraph:
    <input type="checkbox" ng-model="myVar">
  </form>
  <p ng-show="myVar">This is My Pragraph.</p>
  <form>
    Check to show a pragraph:
    <input type="checkbox" ng-model="myCar">
  </form>
  <p ng-show="myCar">My Car name is hunda city.</p>
  <form>
    Check to show a pragraph:
    <input type="checkbox" ng-model="myMob">
  </form>
  <p ng-show="myMob">My Mob name is nokia.</p>
  <form>
    Check to show a header:
    <input type="checkbox" ng-model="myLaptop">
  </form>
  <h3 ng-show="myLaptop">My Lap name is sony.</h3>
</div>

<p>The header's ng-show attribute is set to true when the checkbox is checked.</p>

</body>
</html>

Selectbox

Bind select boxes to your application with the ng-model directive.

The ng-model attribute will have the value of the selected option in the selectbox.

The value of myVar will be either cse, ece, eee or mec .

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body ng-app="">
                    
<form>
Select a Branch of Course:
	<select ng-model="myVar">
	    <option value="cse">CSE
	    <option value="ece">ECE
        <option value="eee">EEE
	    <option value="mec">MEC						
    </select>
</form>

<div ng-switch="myVar">
		<div ng-switch-when="cse">
			<h1>CSE</h1>
			<p>Read about CSE.</p>
		</div>
		<div ng-switch-when="ece">
			<h1>ECE</h1>
			<p>Read about ECE.</p>
		</div>
		<div ng-switch-when="eee">
			<h1>EEE</h1>
			<p>Read about EEE.</p>
		</div>
		<div ng-switch-when="mec">
			<h1>MEC</h1>
			<p>Read about MEC.</p>
		</div>  
</div>
<p>The ng-switch directive hides and shows HTML sections depending on the value of the dropdown list.</p>
</body>
</html>

CSE

The ng-switch directive hides and shows HTML sections depending on the value of the dropdown list.

Application Code

The ng-app directive defines the AngularJS application.

The ng-controller directive defines the application controller.

The ng-model directive binds three input elements to the user object in the model.

The formCtrl controller sets initial values to the take object, and defines the reset() method.

The reset() method sets the user object equal to the take object.

The ng-clickdirective invokes the reset() method, only if the button is clicked.

<!DOCTYPE html>
<html lang="en">
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="myApp" ng-controller="myCtrl">
  <form onvalidate>
    First Name:</br>
    <input type="text" ng-model="user.firstName"></br>
    Last Name:</br>
    <input type="text" ng-model="user.middleName"></br>
     Last Name:</br>
    <input type="text" ng-model="user.lastName">
    <br><br>    
    <button ng-click="reset()">Reset</button>
    <button ng-click="submit()">Submit</button>
  </form>
  <p>Form = {{user}}</p>
  <p>Receipt = {{user}}</p>
  <p>Get = {{user}}</p>
  <p>Take = {{Take }}</p>
</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
    $scope.Get = {firstName:'Suman',middleName:' kumar',lastName:'sharma'};
    $scope.reset = function() {
        $scope.user = angular.copy($scope.Get);
    
    };
    $scope.reset();
    $scope.submit = function() {
        $scope.user = angular.copy($scope.Get);
    };
    $scope.submit();
});
</script>

</body>
