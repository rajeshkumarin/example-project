AngularJS Filters

Filters are used to change modify the data and can be clubbed in expression or directives using pipe character.

AngularJS provides filters to transform data :

currency Format a number to a currency format.

date Format a date to a specified format.

filter Select a subset of items from an array.

json Format an object to a JSON string.

limitTo Limits an array/string, into a specified number of elements/characters.

lowercase Format a string to lower case.

number Format a number to a string.

orderBy Orders an array by an expression.

uppercase Format a string to upper case.


uppercase filter

The uppercase filter format strings to upper case:

Here we've added uppercase filter to print student name in all capital letters.

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="myApp" ng-controller="studentCtrl">
Fst Name: <input type="text" ng-model="firstName">
<br><br>
Mdl Name: <input type="text" ng-model="middleName">
<br><br>
Lst Name: <input type="text" ng-model="lastName">
<p>The name is full:<b> {{ fullName() | uppercase }}</b></p>

</div>

<script>
var app = angular.module('myApp', []); app.controller('studentCtrl', function($scope) {
    $scope.firstName = "Rohit",
    $scope.middleName = "Parshad",
    $scope.lastName = "Kushvaha",
    $scope.fullName = function(){return $scope.firstName+" " + $scope.middleName +" "+ $scope.lastName;};
});
</script>
 
</body>
</html>

The currency Filter

The currency filter is used to format a number as a currency.

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="myApp" ng-controller="costCtrl">
 <input type="digit" ng-model="fee">
<p>Total Amount :<b> {{  fee | currency }}</b></p>

</div>

<script>
angular.module('myApp', []).controller('costCtrl', function($scope) { $scope.fee = 1000;});
</script>

</body>
</html>
