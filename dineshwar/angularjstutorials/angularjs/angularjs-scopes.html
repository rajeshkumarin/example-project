AngularJS Scopes

Scope is a special javascript object which plays the role of joining controller with the views.

The scope is the binding part between the HTML (view) and the JavaScript (controller).

The scope is available for both the view and the controller.

How to Use the Scope?

When you make a controller in AngularJS, you pass the $scope object as an argument:

When adding properties to the $scope object in the controller, the view (HTML) gets access to these properties.

The scope is a JavaScript object with properties and methods, which are available for both the view and the controller.

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="myApp" ng-controller="myCtrl">

<h1>{{carname +" "+ carcol +" "+ carmodule}}</h1>
<p>It's Car Name: <strong>{{carname}}</strong></p>
<p>It's Col Name: <strong>{{carcol}}</strong></p>
<p>It's Mdl Name: <strong>{{carmodule}}</strong></p>

</div>

<script>
var app = angular.module('myApp', []);
app.controller('myCtrl', function($scope) {
    $scope.carname = "Volvo";
     $scope.carcol = "Gray";
      $scope.carmodule = "Vc15er";
});
</script>

<p>The property "carname" was made in the controller, and can be referred to in the view by using the {{ }} brackets.</p>

</body>
</html>

Know Your Scope

It is important to know which scope you are dealing with, at any time.

So knowing your scope is not an issue, but for larger applications

Each list element has access to the current repetition object, in this case a string, which is referred to by using x.

<!DOCTYPE html>
<html>
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
<body>

<div ng-app="myApp" ng-controller="myCtrl">

<ul>
    <li ng-repeat="x in names">{{x.Emil + ','+ x.user + ', ' + x.id}}</li>
</ul>

</div>

<script>
var app = angular.module('myApp', []);

app.controller('myCtrl', function($scope) {
    $scope.names = [{Emil:'dineshwarpaswan@gmail.com',user:'Dineshwar', id:'Ostc12'},
    {Emil:'anupam.kumar@gmail.com',user:'Anupam',id:'Ogrt10'},];
});
</script>

<p>The variable "x" has a different value for each repetition, proving that each repetition has its own scope.</p>

</body>
</html>

Output

  • {{x.Emil + ','+ x.user + ', ' + x.id}}

The variable "x" has a different value for each repetition, proving that each repetition has its own scope.
